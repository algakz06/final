Project Description

You are on the home page of the project from the team "Udaltsy" as part of the BestHack'22 hackathon (Final). We have implemented the platform fin.tech. You can view financial news, buy and sell currencies.

Authorization system

If the user needs to buy or sell currencies, then registration on the platform is mandatory.

News Portal

Available to all users. Everyone can get acquainted with the latest events in the world.

Smart Support

We have implemented our own smart bot on the server, which helps the user to work with the platform. It is located in the lower right corner of the site.
